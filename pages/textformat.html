Halo! Pada halaman kali ini kita akan melakukan Textformatting pada HTML

Ada beberapa yang bisa dilakukan yaitu :

Bold Text Menggunakan tag B

Important Text Menggunakan tag STRONG

Italic Text Menggunakan tag I

Emphasized Text Menggunakan tag EM

Mark Text Menggunakan tag MARK

Small Text Menggunakan tag SMALL

Deleted Text Menggunakan tag DELL

Inserted Text Menggunakan tag INSERT

Subscript Text Menggunakan tag SUB

Superscript Text Menggunakan tag SUP
